## ⚠️ DISCLAIMER
**FOR MEDICAL EDUCATION ONLY - NOT FOR CLINICAL USE**
This system is designed to teach clinical reasoning concepts. It should not be used for actual diagnosis or treatment decisions.

## 🎯 Features

- **6 Specialized AI Agents** working in concert
- **Real-time evidence tracking** with supporting/contradicting evidence
- **Dynamic confidence updating** based on Bayesian principles
- **Cognitive bias detection** (anchoring, confirmation bias, premature closure)
- **Glassmorphism UI** with interactive visualizations
- **WebSocket streaming** for real-time agent thoughts
- **Session persistence** for multi-stage case progression
